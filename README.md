# 🎨 Telegram DALL-E Bot

Телеграм бот для генерации картинок по текстовому описанию используя AI (Stable Diffusion через Hugging Face).

## ✨ Возможности

- 🎨 Генерирует картинки по описанию
- 🚀 Быстро работает
- 💰 Полностью бесплатно
- 🌍 Поддерживает русский и английский языки
- ⚡ Легко развернуть

## 📋 Требования

- Python 3.8+
- Аккаунт Telegram (для создания бота)
- Бесплатный аккаунт Hugging Face

## 🚀 Быстрый старт

### 1️⃣ Получи токены

**Telegram токен:**
1. Открой Telegram и найди **@BotFather**
2. Напиши `/newbot`
3. Дай имя боту: "AI Художник"
4. Дай юзернейм: `ai_artist_bot_2024` (или любой другой)
5. **Скопируй токен** (выглядит как: `123456:ABC-DEF1234...`)

**Hugging Face токен:**
1. Зайди на https://huggingface.co
2. Зарегистрируйся (бесплатно)
3. Перейди в Settings → Access Tokens
4. Создай новый token: "Create new token"
5. **Скопируй токен**

### 2️⃣ Локальная установка

```bash
# Клонируй репозиторий
git clone https://github.com/nikvik1590-creator/telegram-dalle-bot.git
cd telegram-dalle-bot

# Установи зависимости
pip install -r requirements.txt

# Создай файл .env
cp .env.example .env
```

### 3️⃣ Заполни .env

Открой файл `.env` и вставь токены:

```env
TELEGRAM_TOKEN=твой_токен_от_BotFather
HUGGINGFACE_TOKEN=твой_токен_от_Hugging_Face
```

### 4️⃣ Запусти бота

```bash
python bot.py
```

Открой Telegram → найди своего бота → напиши `/start`

## 🐛 Проблемы?

### "ModuleNotFoundError: No module named 'telegram'"
```bash
pip install -r requirements.txt
```

### "ошибка при создании картинки"
- Проверь, что токен Hugging Face правильный
- Может быть очередь на сервере - подожди и попробуй снова
- Попробуй более короткое описание

### "Бот не отвечает"
- Проверь, что `TELEGRAM_TOKEN` правильный
- Перезапусти бота: `Ctrl+C` и `python bot.py`
